Place hero images here to use them in the site's HeroCarousel.

Filenames (required):
- hero-1.jpg (or hero-1.jpeg)
- hero-2.jpg (or hero-2.jpeg)
- hero-3.jpg (or hero-3.jpeg)

Recommended size: 2000–2500px wide (landscape), e.g. 2070×1200 or 2400×1400.
Use high-quality JPG or WebP. After placing the files, run the dev server (`npm run dev`) and check the homepage.

If you want different filenames, update `app/components/HeroCarousel.tsx` and restart the dev server.
